# Konzept und Einleitung

## Einleitung
Heutzutage muss Wissen ohne großen Aufwand gespeichert und sehr schnell verändert werden können. Künstliche Intelligenz kann dabei ebenfalls unterstützen und die Bearbeitung verbessern. Viele Personen können Inhalte bearbeiten, der Admin kontrolliert die geschriebenen Inhalte. Er kann Korrekturen vornehmen und freigeben. Mein Projekt speichert keine Benutzerdaten. Aus diesem Grund kann ich meine Inhalte nicht direkt bearbeiten; andere Nutzer können sie jedoch herunterladen und weiterbearbeiten
