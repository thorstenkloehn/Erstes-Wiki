# Architektur

Dieses Projekt basiert auf ASP.NET Core und verwendet das MVC-Muster.

- **Controller**: Steuern die Logik und die Navigation.
- **Models**: Repräsentieren die Datenstruktur.
- **Views**: Präsentieren die Daten im Browser.

Die Authentifizierung erfolgt über ASP.NET Identity.
Die Datenbank ist SQLite.
